# @yllo/ui-lib

[![yllo/ui-lib NPM package](https://img.shields.io/npm/v/@yllo/ui-lib?color=green)](https://www.npmjs.com/package/@yllo/ui-lib)

![Yllo OS](https://yllo.co/ui/assets/images/yos-logo-dark.svg)

[Yllo OS](https://yllo.co/about) component library

## Installation instructions

```js
npm install @yllo/ui-lib
```

## Start using the library

```jsx
<script>
  import { onMount } from 'svelte'
  import ui, { EThemeType, Button, EColor } from '@yllo/ui-lib'

  onMount(() => {
    ui.setThemeOptions({
        themeType: EThemeType.light,
        isRounded: true,
        isShadow: true,
        isBorder: true,
        isInteractiveCursor: true,
        isCircleCursor: true,
    })
  })
</script>

<Button color={EColor.line7}>Hello world!</Button>

```
