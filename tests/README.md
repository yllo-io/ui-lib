
## Local Launch UI-lib & Dev config

### Local link

1. Open package folder

```
cd package
```

2. Install & Local Link

```
npm install
npm link
```

### Launch

1. Open tests folder

```
cd tests
```

2. Install

```
npm install
```

3.  Local Link @yllo/ui-lib

```
npm link @yllo/ui-lib
```

4. Start dev server
   (localhost:8080)

```
npm run dev
```
